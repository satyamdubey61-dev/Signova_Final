# SIGNOVA

AI Powered Indian Sign Language Translation System.

## Features
- Real-time sign language detection
- CNN-based gesture recognition
- Voice output
- Translation support
- Flask web application

## Technologies Used
- Python
- Flask
- OpenCV
- TensorFlow/Keras
- CNN
- HTML/CSS/JavaScript

## Project Structure
- core/
- models/
- routes/
- services/
- templates/
- static/

## Installation

pip install -r requirements.txt

python app.py

## Future Scope
- More gesture support
- Mobile app
- Cloud deployment
- Faster real-time detection
